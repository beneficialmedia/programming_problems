Programming Problem Solutions

  1. A program that prints ‘Hello World’ to the screen.
  2. A program that asks for your name and greets you with your name.
  3. The previous program (modified), such that only the people Alice and Bob are greeted with their names.
  4. A program that asks for a number n and prints the sum of the numbers 1 to n.
  5. The previous program (modified), such that only multiples of three or five are considered in the sum, e.g. 3, 5, 6, 9, 10, 12, 15 for n = 17.
  6. A program that asks the user for a number n and gives people the possibility to choose between computing the sum and computing the product of 1, …, n.
